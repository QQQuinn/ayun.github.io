只狼

发表于 2020-05-28 分类于 游戏记录

与魂系列的一贯传统一般游戏的魅力就是打败较高难度的敌人,自身获得相应的成就已经满足感
游戏剧情
故事的起源即樱龙降临苇名源之宫，并带来不死的力量源之宫内的所有生物都被感染成非人的生物，随之感染的还有湖内的鲤鱼王湖与水生村相同，水生村随即也变得即为诡异，水流一路而下感染到仙峰寺附近的白猿，仙峰寺一位僧人追求不死，接受了虫子的寄生，成为了源之宫的守门人…………苇名一心掀起盗国战争，击败田村。故事发生在二十多年后的苇名，苇名一心老去病重。国家败颓屑一郎研究不死的力量，主角狼保护皇子，故事就此展开详细剧情可以去看狼学家分析，此处不多说游戏在初期会获得一个”铃铛”的道具至此之后游戏分为两条线路，一条为回忆路线，一条为现实路线有趣的是回忆路线中做的事情会影响到现实路线中的变化

战斗系统
攻击方式:普通、投技、横扫、突刺
招架:在敌方即将攻击到自身的时候成功使用招架，可防御此次攻击,会大幅增加自身的架势槽、可以防御普通攻击若完美招架，则小幅增加自身的架势槽，且不会溢出，增加对方招架条，完美招架可以防御普通、突刺架势槽满值时，露出破绽，此时可进行忍杀
闪避:对敌方的攻击进行闪避，闪避的好可防御一切攻击。不过最常用来防御投技，只能用闪避来防御投技如果学习了看破技能，对敌方突刺攻击使用闪避可看破敌方攻击，大量增加敌方架势槽
跳跃:克制横扫攻击，敌方使用横扫时，使用两次跳跃可以踩头，大量增加敌方架势槽
忍义手:战斗额外攻击方式，专属于主角，对战斗的帮助非常之大。分为鞭炮，手里剑，斧头，长枪，伞，喷火器，神隐，毒刀，雾鸦，口哨一般的敌人都会有对应忍义手的弱点，游戏中会出现较为隐晦的提示，玩家如果仔细查找，提前获知，战斗事半功倍
忍杀:在敌人未察觉或架势条为空时，可进行忍杀，直接秒杀敌方
